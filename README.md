## 📖 About

4K Image Compressor 1.4.0.0220 is a standalone Windows utility designed for fast, lossy and lossless compression of high-resolution images up to 4K quality. It reduces file size without drastically affecting visual fidelity, making it ideal for web optimization, storage reduction, and sharing. The tool processes images in batch mode and supports common formats including JPEG, PNG, and WebP. No cloud uploads are required—everything runs locally on your machine.
